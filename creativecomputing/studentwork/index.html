Midterm Assignments

This assignment is the seventh of seven related assignments that build in complexity to introduce concepts of programming, data structures, logic, functions, loops, the DOM, events, and libraries. Taking inspiration from The Name Game, build an app that takes any name as input (assuming that the name is a single first name, and that it is capitalized). The app should strictly follow the rules of The Name Game.

Final Assignments

Your submission for the final assignment will include work in HTML, CSS, and JavaScript to create a final web-based composition that exhibits the knowledge you have gained over the semester. You will also be adding it to your learning portfolio. The assignment should have strong visual and interactive components. It should include elements of input/output, response, logic, data storage/access, and iteration.
